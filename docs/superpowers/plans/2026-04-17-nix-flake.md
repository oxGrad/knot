# Nix Flake Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `flake.nix` that provides a reproducible dev shell, a `nix build` target for the knot binary, and three distro isolation test commands (`nix run .#test-ubuntu/fedora/arch`) that layer knot onto real Ubuntu/Fedora/Arch base images.

**Architecture:** The flake exposes outputs for `x86_64-linux` and `aarch64-darwin` via `forAllSystems`. The knot binary is always cross-compiled to linux/amd64 using `pkgsCross.gnu64` so it can run inside Linux containers regardless of host arch. Base distro images are pulled by pinned SHA256 digest via `dockerTools.pullImage`, then `knot` is copied in via `buildLayeredImage`. Test apps are shell scripts that call `nix build`, `docker load`, and `docker run` in sequence.

**Tech Stack:** Nix flakes, nixpkgs `buildGoModule`, `dockerTools.pullImage`, `dockerTools.buildLayeredImage`, `nix-prefetch-docker`

## Task 1: Bootstrap flake with dev shell

**Files:**
- Create: `flake.nix`

- [ ] **Step 1: Create `flake.nix` with dev shell only**

```nix
{
  description = "knot — a dotfile manager";

  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
  };

  outputs = { self, nixpkgs }:
    let
      systems = [ "x86_64-linux" "aarch64-darwin" ];
      forAllSystems = nixpkgs.lib.genAttrs systems;
    in {
      devShells = forAllSystems (system:
        let pkgs = nixpkgs.legacyPackages.${system};
        in {
          default = pkgs.mkShell {
            buildInputs = with pkgs; [ go golangci-lint goreleaser git ];
            shellHook = ''
              export GOPATH="$PWD/.nix-go"
              export GOMODCACHE="$PWD/.nix-go/pkg/mod"
              echo "knot dev shell — $(go version)"
            '';
          };
        });
    };
}
```

- [ ] **Step 2: Generate the lock file**

```bash
nix flake lock
```

Expected: `flake.lock` created in the project root.

- [ ] **Step 3: Verify dev shell works**

```bash
nix develop
```

Expected: shell prompt changes, `go version` prints a Go version, `golangci-lint --version` works, `goreleaser --version` works. Type `exit` to leave.

- [ ] **Step 4: Commit**


## Task 5: Add test apps

- [ ] **Step 1: Add `apps` output to `flake.nix`**

Add this block to the `outputs` return attrset, after `packages`:

```nix
      apps = forAllSystems (system:
        let
          pkgs = nixpkgs.legacyPackages.${system};
          mkTestApp = distro:
            let
              script = pkgs.writeShellScript "knot-test-${distro}" ''
                set -euo pipefail
                echo "==> Building knot-${distro} image..."
                out=$(nix build ".#images.${distro}" --no-link --print-out-paths)
                echo "==> Loading into Docker..."
                docker load < "$out"
                echo "==> Running knot-${distro} (exit or Ctrl-D to quit)..."
                exec docker run --rm -it "knot-${distro}:latest" "$@"
              '';
            in { type = "app"; program = "${script}"; };
        in {
          test-ubuntu = mkTestApp "ubuntu";
          test-fedora = mkTestApp "fedora";
          test-arch   = mkTestApp "arch";
        });
```
